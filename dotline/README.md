## 线条粒子效果插件
> powered by Cecil

### Get Started
> new DotLine($id).init()
Or
> new DotLine({
    id: 'canvas', // default 'canvas'
    option: {
        ball_speed_range: [0.5,1.3],
        line_width: 0.08,
        ball_count: 15,
        canvas_height: 400
    }
})
